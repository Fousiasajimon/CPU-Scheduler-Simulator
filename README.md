# CPU-Scheduler-Simulator
A high-performance, web-based simulator designed to visualize and analyze CPU scheduling algorithms. This tool provides a real-time graphical representation of process execution, allowing users to compare efficiency across different scheduling strategies.

Features:
Dynamic Gantt Chart: Visualizes execution & idle time.
Virtual Keypad: Seamless data entry for AT, BT, and Priority.
Calculations: Real-time CT, TAT, WT, and Averages.
Custom Logic: Toggle priority direction.
